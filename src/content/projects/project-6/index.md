### Politics Strategy Game - Game Development

![Politics Game Screenshot](/politics-game-screenshot.png)
*Screenshot from the prototype*

**Politics** is an ambitious modular strategy game that challenges traditional 4X game conventions by allowing players to build empires with mixed cultural elements. As an **unpaid programming intern** on this three-person development team, I used this opportunity to learn Rust and Bevy Engine while contributing to the project.

#### My Role as Intern:
- **Learning Rust**: Gained hands-on experience with Rust programming language through real-world game development
- **Bevy Engine Exploration**: Learned data-driven game engine architecture and ECS (Entity Component System) patterns
- **Debugging Support**: Helped identify and fix bugs across various game systems
- **Feature Development**: Implemented small features and improvements under guidance from senior developers
- **Code Review**: Participated in code review process to learn best practices and improve code quality

#### Current Status:
The project is in active development with a playable prototype available.

This internship opportunity provided invaluable hands-on experience with cutting-edge technologies while contributing to an innovative game project. It demonstrates my commitment to continuous learning and ability to adapt to new technologies in a collaborative development environment.
